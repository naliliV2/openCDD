Event handler du composant.

Cette fonction est appelée lorsque le bot reçoit un évènement.

Si l’event s’est bien passé ou n’a pas été traité, elle doit retourner Ok(()). Sinon, un Err contenant le message d’erreur doit être retourné. Ce message d’erreur sera ensuite renvoyé à la sortie standard.

Retournes le groupe de commandes lié au composant.

Le système d’aide du bot se repose sur ce groupe. Vu que le parse de la commande n’est pas obligatoire, cette fonction est donc optionnelle.
